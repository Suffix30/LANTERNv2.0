# Agent BLACK

AI-powered security companion for LANTERN.

## Quick Start

```bash
pip install "lantern-scanner[agent] @ git+https://github.com/Suffix30/LANTERNv2.0.git"
black status
black chat
```

## Features

- Natural language scanning via LANTERN
- Situational awareness (Overwatch mode)
- Autonomous pentesting with AI planning
- Knowledge base with RAG
- Remote tool execution on Kali hosts
- Obsidian vault integration

## Requirements

- Python 3.10+
- LANTERN installed
- Ollama or local GGUF model (for AI features)
